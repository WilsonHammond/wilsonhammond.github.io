# Microsteps For GitHub And LinkedIn ## Part 1, Create The GitHub Repository 1. Open GitHub. 2. Click your profile picture. 3. Click Your repositories. 4. Click New. 5. Repository name: project_violet_ledger 6. Description: Third party Governance, Risk, and Compliance lab focused on vendor risk, human controls, and enterprise recovery. 7. Set visibility to Public. 8. Check Add a README file only if you are not uploading the README from this package. 9. Click Create repository. ## Part 2, Upload The Lab Files 1. Open the new repository. 2. Click Add file. 3. Click Upload files. 4. Drag the contents of this package into the upload window. 5. Keep the folder structure intact. 6. Commit message: Initial upload for Project Violet Ledger GRC lab 7. Click Commit changes. ## Part 3, Publish The GitHub Pages Site 1. Open the repository. 2. Click Settings. 3. Click Pages. 4. Under Build and deployment, choose Deploy from a branch. 5. Branch: main 6. Folder: /site 7. Click Save. 8. Wait until GitHub shows the site link. 9. Open the link and confirm the page loads. ## Part 4, Add This Project To Your Existing Portfolio Site 1. Open your main GitHub Pages repository. 2. Click Add file. 3. Click Upload files. 4. Upload the site folder contents into a new folder named violet_ledger. 5. Link to it from your main portfolio page. 6. Use this project title: Project Violet Ledger 7. Use this short description: Third party vendor risk and human control effectiveness lab built around a simulated healthcare analytics breach. ## Part 5, Post To LinkedIn 1. Open LinkedIn. 2. Click Start a post. 3. Paste the LinkedIn post from docs/linkedin_post.md. 4. Add the GitHub repository link. 5. Add the GitHub Pages project link. 6. Add hashtags only at the bottom. 7. Recommended hashtags: #GRC #Cybersecurity #RiskManagement #ThirdPartyRisk #InformationSecurity #GovernanceRiskCompliance 8. Post after the GitHub link works. ## Part 6, Add To LinkedIn Featured Section 1. Go to your LinkedIn profile. 2. Find Featured. 3. Click the plus button. 4. Choose Add a link. 5. Paste the GitHub Pages project link. 6. Title: Project Violet Ledger 7. Description: Third party vendor risk lab with risk register, human control assessment, control mapping, and remediation roadmap. 8. Click Save. ## Part 7, Add To LinkedIn Projects Section 1. Open your LinkedIn profile. 2. Click Add profile section. 3. Click Additional. 4. Click Add projects. 5. Project name: Project Violet Ledger 6. Description: Built a simulated Governance, Risk, and Compliance case study analyzing third party vendor compromise, human control degradation, residual risk, cost exposure, and executive remediation strategy. 7. Link: Paste the GitHub Pages project link. 8. Click Save.
