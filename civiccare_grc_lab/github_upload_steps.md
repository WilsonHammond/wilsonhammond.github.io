# GitHub Upload Steps

## Goal

Add this lab to your GitHub Pages repository or create a separate repository called grc_portfolio_labs.

## Recommended Folder Name

civiccare_grc_lab

## Files to Upload

1. README.md  
2. scenario.md  
3. data_classification.md  
4. risk_register.csv  
5. control_crosswalk.csv  
6. evidence_request_list.md  
7. poam.md  
8. executive_summary.md  
9. interview_talk_track.md  
10. linkedin_post.md  
11. sources.md

## Manual Upload Path

1. Open GitHub.  
2. Open your portfolio repository.  
3. Select Add file.  
4. Select Upload files.  
5. Drag the civiccare_grc_lab folder contents into GitHub.  
6. Commit with this message: Add CivicCare Nexus GRC portfolio lab.  
7. After upload, open README.md to confirm the project displays correctly.

## Portfolio Page Summary

Use this summary on your website project card.

CivicCare Nexus is a fictional Governance, Risk, and Compliance lab examining how a health technology company should manage regulated data across healthcare, finance, and federal program support. The project includes a risk register, control crosswalk, evidence request list, Plan of Action and Milestones, executive summary, and interview talk track. It demonstrates practical understanding of data classification, control gaps, evidence, remediation, and residual risk.
